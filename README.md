# WazimoAssignment

# Requirements

Please write a single page application that receive data from https://storage.googleapis.com/static.aoni.io/demo/user.json and show it on the grid.

1- Retrieve all data form server end point https://storage.googleapis.com/static.aoni.io/demo/user.json

2- Show data in grid

3- Enable text filtering on columns

4- Push your code into github & share it with us

5- Min Requirement: Angular 6
